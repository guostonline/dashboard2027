MÉTHODOLOGIE DES CALCULS

Objectif Mois (OBJ MOIS) : Établi au prorata du nombre de jours total de l'agence.
OBJ MOIS = Objectif de base × Jours Ouvrés (24) / Total Jours dans le mois (depuis l'Excel)

Reste à faire (RAF) : Montant quotidien restant pour atteindre l'objectif ajusté.
RAF = (Objectif Mois - Réalisé avec Encours) / Jours Restants (Configurés ci-dessus)

Taux ACM (Achat Client Moyen) : Pourcentage d'efficacité de commande.
ACM = Clients Facturés / Clients Programmés

RAF TSM / ACM : Nombre de visites/achats quotidiens requis.
RAF = (Clients Programmés - Réalisé) / Jours Restants
